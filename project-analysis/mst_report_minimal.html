01 Background

Episodic memory depends on hippocampal pattern separation — the ability to orthogonalise overlapping input representations into distinct neural codes, mediated by the dentate gyrus (DG) and CA3 (Yassa & Stark, 2011). The Mnemonic Similarity Task (MST) provides a validated behavioural measure of this capacity: participants encode everyday object and scene images, then classify test items as Old, Similar, or New — including high-interference lure images at five graded similarity levels (Stark et al., 2013).

Event Segmentation Theory (Zacks et al., 2007) proposes that experience is chunked into discrete events at event boundaries — moments of contextual change where working memory updates its situational model. Morse et al. (2023) showed these boundaries directionally influence item memory: post-boundary items show recognition deficits; pre-boundary items show enhanced mnemonic discrimination.

The present study replicates and extends Morse et al. across three between-subjects conditions, with 40 encoding events × 7 items each, a modified orienting task ("Will it fit in a shoebox?"), and explicit lure-bin tracking (bins 1–5).

Event Position

Within each 7-item encoding event: serial position 1 = post-boundary; position 7 = pre-boundary; positions 2–6 = mid-event. Position is inferred from the order of image_path rows in the task CSV.

Key Metrics

REC = P("old" | Target) − P("old" | Foil)
Recognition Memory Index — bias-corrected hit rate
LDI = P("similar" | Lure) − P("similar" | Foil)
Lure Discrimination Index — hippocampal pattern separation proxy

03 Research Hypotheses

H1 — Post-boundary ↓ REC
Recognition Deficit

Post-boundary target REC will be significantly below zero and below mid-event REC. Event transitions reset the encoding context.

H2 — Pre-boundary ↑ LDI
Discrimination Enhancement

Pre-boundary LDI will exceed post-boundary LDI. Heightened attention at event closure strengthens high-fidelity encoding.

H3 — Lure Bin Gradient
Monotonic LDI decrease

LDI decreases monotonically from Bin 1 (most similar) to Bin 5 (least similar), reflecting graded pattern separation demand.

H4 — Condition Moderation
Group × Position Interaction

Boundary effects on REC and LDI differ across conditions; "both" condition expected to show the strongest effects.

Exploratory: RT elevation at post-boundary; object vs. scene stimulus differences; LDI bin × position interaction.

04 Methods

Preprocessing Pipeline

  1. Load task CSV → extract image_path order → assign pre/mid/post position (7-item window).
  2. Unify RT: use key_resp_9.rt if available; else total RT = 3 + key_resp_8.rt.
  3. Load test CSV → classify trials (target/lure/foil by filename suffix) → map response keys to old/similar/new.
  4. Cross-reference Set6 bins.txt / SetScC bins.txt to assign lure bins 1–5.
  5. For item_only: recalculate encoding accuracy using scenes_mapping.txt to correct forced-choice errors.
  6. Compute foil baseline rates → calculate per-participant REC and LDI by position and by lure bin.

05 Anticipated Results

H1 · REC Post Post-boundary REC expected significantly below zero: one-sample t(~159) < −2.0, p < 0.0125. Paired comparison post vs pre: Cohen's d ≈ −0.30 to −0.45.
H2 · LDI Pre Pre-boundary LDI significantly above zero: t(~159) > 2.0, p < 0.0125. Pre > Post LDI: d ≈ 0.25–0.40.
H3 · Bin gradient ANOVA across bins 1–5: F(4, ~636) > 3.5, p < 0.01, η² > 0.05. Linear trend contrast significant.
H4 · Condition Condition × Position interaction in mixed ANOVA tests whether boundary effects generalise. "Both" condition expected to show largest d for pre−post REC difference.
Exploratory · RT Median RT elevated post-boundary relative to pre-boundary. Aligns with EST predictions of increased context-updating demands at boundaries.

07 Conclusion & Next Steps

This study provides a three-condition replication of Morse et al. (2023) with a substantially larger N (~160), five explicit lure-bin levels, and a novel orienting task. The LDI metric's sensitivity to lure similarity gradients makes it particularly well-suited to detecting subtle encoding precision differences across event positions.

  • Complete preprocessing and run mst_analysis.py on full dataset.
  • Pre-register confirmatory hypotheses on OSF before examining group-level results.
  • Investigate individual differences: encoding task accuracy as a moderator of boundary effects.
  • Supplement with fMRI/EEG to link behavioural LDI to DG/CA3 BOLD activity at boundaries.
  • Extend to aging/MCI populations where hippocampal pattern separation is compromised.
