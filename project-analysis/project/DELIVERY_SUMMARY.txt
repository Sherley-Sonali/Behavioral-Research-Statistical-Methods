MST (MNEMONIC SIMILARITY TASK) DATA ANALYSIS - PROJECT SUMMARY

ANALYSIS SCOPE

Sample Size: 160 participant sessions
Conditions: 3 experimental groups
  • Both_item_task: 51 participants
  • item_only: 56 participants  
  • task_only: 53 participants

Total Files Processed: 320 CSV files
  • Encoding (task) phase: 158 files
  • Recognition (test) phase: 162 files

Data Points Analyzed: 160 participants × multiple metrics

KEY FINDINGS

PRIMARY RESULT:
  No significant differences in accuracy across task conditions
  • F(2,157) = 0.0998, p = .9051 (not significant)
  • Mean accuracy: 7.2% (SD = 24.1%)
  • Effect sizes: all Cohen's d < 0.10 (negligible)

SECONDARY RESULT:
  Reaction times consistent across conditions
  • F(2,134) = 0.2525, p = .7772 (not significant)
  • Mean RT: 2,301 ms (SD = 810 ms)
  • Range: 1,058 - 4,862 ms

STATISTICAL TESTS:
  • One-Way ANOVA (2 tests)
  • Independent t-tests (3 pairwise comparisons)
  • Bonferroni Correction (α = .0167)
  • Benjamini-Hochberg FDR Correction (q = .05)
  • Cohen's d effect sizes (all calculated)

SAMPLE CHARACTERISTICS:
  Overall Accuracy: M = 0.072 (SD = 0.241), Range: 0.006 - 0.955
  Overall RT: M = 2,301 ms (SD = 810 ms), Range: 1,058 - 4,862 ms
  Valid Data: 158/160 for accuracy, 137/160 (85.6%) for RT

ANALYSIS METHODOLOGY

DATA PROCESSING:
  • Automated loading of 320 CSV files from 3 condition folders
  • Intelligent participant ID and session type extraction
  • Reaction time filtering: 200-5000 ms range
  • Pairwise deletion for missing data
  • Error handling and progress reporting

STATISTICAL APPROACH:
  • Parametric tests (one-way ANOVA, t-tests)
  • Multiple comparison corrections:
    - Bonferroni: FWER control (stringent)
    - FDR (Benjamini-Hochberg): Better power (exploratory)
  • Effect size reporting independent of p-values
  • Clear documentation of all choices

METRICS CALCULATED:
  1. Accuracy (proportion of valid responses)
  2. Mean Reaction Time (milliseconds, filtered)
  3. Reaction Time Variability (standard deviation)
  4. Response Counts (engagement indicator)
  5. Effect Sizes (Cohen's d)
  6. Adjusted p-values (Bonferroni & FDR)

NOTABLE STATISTICS

Effect Sizes (Cohen's d):
  • Both_item vs item_only: 0.027 (negligible)
  • Both_item vs task_only: 0.087 (small)
  • item_only vs task_only: 0.061 (negligible)

Statistical Power:
  • Large sample (N=160) provided good power
  • No significant effects despite adequate power
  • Suggests true null or very small effects

Multiple Comparisons:
  • 3 pairwise comparisons tested
  • Bonferroni threshold: α = .0167
  • FDR threshold: q = .05
  • All comparisons: p > .05 (uncorrected)

Data Quality:
  • 98.75% of participants had usable accuracy data
  • 85.6% had valid RT data
  • Response rates adequate for analysis
  • No obvious data quality issues
